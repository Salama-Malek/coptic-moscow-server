# Admin Panel User Guide

## Logging In

1. Open your browser and go to: `https://coptic-api.sm4tech.com/admin`
2. Enter your email and password
3. On first login, you will be prompted to change the temporary password

## Sending an Announcement Using a Template

1. In the sidebar, click **New Announcement**
2. Select the appropriate template from the dropdown (e.g., "Saturday Vespers + Sunday Liturgy")
3. Fill in the required fields (dates, times, etc.)
4. A live preview of the text will appear on the right
5. Select **Priority**: Normal (no special sound), High (default sound), Critical (bell sound even on silent)
6. Select **Category**: Service or Announcement
7. Choose **Send now** or **Schedule for** and set the time
8. Click **Preview & Send**
9. Review the text in the confirmation dialog and click **Confirm**

## Sending a Custom Announcement (No Template)

1. Click **New Announcement**
2. Select **No template** from the dropdown
3. Type the title in Arabic (required) and optionally in Russian and English
4. Write the announcement body
5. Follow the same steps as above

## Managing the Calendar

1. In the menu → **Calendar**
2. To add a new event: click **Add Event**
3. Enter the title in Arabic, Russian, and English
4. Set the start date and time
5. Choose recurrence: none, daily, weekly, monthly, yearly
6. Set the duration and reminder time
7. Click **Save**

### Moveable Feasts
Moveable events (Great Lent, Jonah Fast, Apostles' Fast) are pre-loaded but inactive. To activate them:
1. Click **Edit** next to the event
2. Set the correct start date for this year
3. Toggle **Active** on
4. Click **Save**

## Adding a New Admin

1. In the menu → **Team** (available only to super admins)
2. Click **Add Admin**
3. Enter the name, email, and select a role
4. A temporary password will appear — **save it immediately**, it will not be shown again
5. Give the temporary password to the new admin for their first login

## Changing Your Password

1. In the menu → **My Account**
2. Enter your current password and new password
3. Click **Save**

## Important Notes

- All sent announcements are logged and cannot be deleted after sending
- Drafts and scheduled announcements can be deleted before they are sent
- Changing the language in the top bar only changes the admin panel language, not the language of sent announcements
